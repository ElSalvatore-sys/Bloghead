# Bloghead - Demo Anleitung

## Vorbereitung (1 Minute)

1. Browser oeffnen (Chrome oder Firefox)
2. Gehe zu: **http://localhost:5173**
3. Stelle sicher: Dev Server laeuft (`npm run dev`)

---

## Demo-Ablauf (15 Minuten)

### Teil 1: Startseite zeigen (2 Min)

**Klicke/Zeige:**
1. Logo oben links
2. Scrolle langsam nach unten
3. Zeige: Hero-Bereich mit Animation
4. Zeige: Feature-Karten
5. Zeige: Kuenstler-Vorschau
6. Zeige: Footer mit Links

**Sage:**
> "Das ist die Startseite. Modern, animiert, komplett auf Deutsch."

---

### Teil 2: Kuenstler durchsuchen (3 Min)

**Klicke:**
1. "Kuenstler" im Menue oben
2. Zeige das Kuenstler-Grid

**Demonstriere:**
1. Genre-Filter auswaehlen (z.B. "DJ")
2. Standort eingeben
3. Preis-Slider bewegen

**Sage:**
> "Veranstalter koennen hier Kuenstler nach Genre, Ort und Preis filtern."

---

### Teil 3: Kuenstler-Profil (3 Min)

**Klicke:**
1. Auf eine Kuenstler-Karte klicken

**Zeige:**
1. Grosses Profilbild
2. Name und Genre
3. Bewertungssterne
4. "Ab XX EUR" Preis
5. Verfuegbarkeitskalender (gruen = frei)

**Klicke:**
1. "Buchen" Button
2. Zeige das Buchungsformular

**Sage:**
> "Mit einem Klick kann man eine Buchungsanfrage senden."

---

### Teil 4: Events (2 Min)

**Klicke:**
1. "Events" im Menue

**Zeige:**
1. Event-Karten mit Datum
2. Location-Info
3. Preis/Eintritt

---

### Teil 5: Registrierung (2 Min)

**Klicke:**
1. "Anmelden" Button oben rechts
2. Dann "Registrieren" Link

**Zeige:**
1. Schritt 1: Benutzertyp waehlen (Fan, Kuenstler, etc.)
2. Schritt 2: E-Mail und Passwort
3. Schritt 3: Profil-Details
4. Google/Facebook Buttons

**Sage:**
> "Die Registrierung ist in 3 einfache Schritte unterteilt."

---

### Teil 6: Mobile Ansicht (1 Min)

**Mache:**
1. Browser-Fenster kleiner ziehen

**Zeige:**
1. Hamburger-Menue erscheint

> "Die Website funktioniert auf jedem Geraet - Desktop, Tablet, Smartphone."

---

### Teil 7: Admin-Bereich (2 Min)

**Gehe zu:**
1. /admin (direkt in URL eingeben)

**Zeige:**
1. Dashboard mit Statistiken
2. Benutzerliste
3. Meldungen
4. Ankuendigungen

**Sage:**
> "Der Admin sieht alles auf einen Blick und kann die Plattform verwalten."

| Problem | Loesung |
|---------|---------|
| Seite laedt nicht | `npm run dev` neu starten |
| Styling fehlt | Strg+Shift+R (Cache leeren) |
| Fehler im Browser | "Das ist ein Entwickler-Feature" |
